PROBLEM

Diabetic Retinopathy is a common complication of diabetes mellitus, which causes lesions on the retina that affect vision. If it is not detected early, it can lead to blindness.Unfortunately ,DR is not a reversible process,and treatment only sustains vision.DR early detection and treatment can significantly reduce the risk of vision losss. the manual diagnosis process of DR retina fundus images by opthalmologists is time,effort and cost consuming and pronr to misdiagnosis unlike computer aided detection.

SOLUTION

In this project we will be building a tranfer learing model that can detect and classify types of diabetic retinopathy.A web application is integrated with a model from where the user can upload a Diabetic retinopathy (DR) image like Mild DR,Severe DR ,etc.. and see the analysed result on user interface.
